## Source

* **Code**

  * demo.py：呈現Demo的結果（對6次投籃進行軌跡偵測與判斷進球）

  * test.py：呈現evaluation的結果（對128次投籃進行軌跡偵測與判斷進球）

  * background.py：呈現report中提到的background subtractionc後的結果（6次投籃的影片）

* **Video**

  * demo.mp4：進行6次投籃的影片
  * test.mp4：進行128次投籃的影片

* **How to run code**

  * Required package

    ```pip3 install cv2```

    ```pip3 install imutils```

  * Run code

    ```python demo.py```

    ```python test.py```
